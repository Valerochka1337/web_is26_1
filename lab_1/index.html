Projects

Project 1: Task Management System

This project is a task management system designed for software development teams. It allows users to create, assign, and track tasks, as well as manage projects and teams. The system provides functionality for commenting on tasks and notifying users of important events.

  • Spring Boot: Main framework for creating RESTful services.
  • Spring Data JPA: For database interaction.
  • Spring Security: For managing authorization and authentication.
  • Hibernate: For ORM (Object-Relational Mapping).
  • MySQL: As the database.
  • Thymeleaf: For creating the web interface.
  • Swagger: For API documentation.

Project 2: E-commerce Platform

This project is an e-commerce platform where users can browse products, add them to their cart, and make purchases. Administrators can manage the product catalog, view orders, and handle inventory. The platform supports multiple payment methods and integration with various shipping services.

  • Spring Boot: For API development.
  • Spring MVC: For building the web interface.
  • Spring Data JPA: For database interactions.
  • Spring Security: For user and role management.
  • H2: Lightweight embedded database for development and testing.
  • Freemarker: For HTML generation.
  • RabbitMQ: For managing order and notification queues.
  • PayPal API: For handling payments.

Project 3: Hotel Booking System

The hotel booking system allows users to search for and book rooms in hotels. Hotel administrators can manage available rooms, prices, and special offers. The platform supports reviews and ratings and integrates with external services for sending confirmations and reminders.

  • Spring Boot: For developing the booking service.
  • Spring Data JPA: For managing booking data.
  • Spring Security: For user authentication and authorization.
  • PostgreSQL: As the main database.
  • Thymeleaf: For rendering pages.
  • Docker: For containerizing the application.
  • Twilio API: For sending SMS notifications.

Project 4: Online Course Platform

This platform allows users to register and take online courses. Instructors can create courses, upload videos, materials, and assignments, and track student progress. The system supports discussion forums and an assignment submission and grading system.

  • Spring Boot: For building the core business logic.
  • Spring Data JPA: For database interactions.
  • Spring Security: For managing access and roles (students, instructors, administrators).
  • MongoDB: For storing course materials and student progress data.
  • Angular: For creating a rich frontend.
  • WebSocket: For real-time updates and forum discussions.
  • AWS S3: For storing video materials.
